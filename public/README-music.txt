# Wedding Music Instructions

Để thêm nhạc cưới cho website:

1. Đặt file MP3 có tên 'wedding-music.mp3' vào thư mục public/
2. File nhạc sẽ tự động được phát khi người dùng nhấn nút play

Nguồn nhạc cưới miễn phí:
- https://freemusicarchive.org/
- https://incompetech.com/
- https://www.bensound.com/
- https://pixabay.com/music/

Lưu ý: File nhạc nên có dung lượng nhỏ (< 5MB) để tải nhanh.
